👉 https://hablandodedata.substack.com

Aquí encontrarás ejemplos prácticos de:

- Databricks
- Data Engineering
- Lakehouse & Lakeflow
- Data Governance

Todos los ejemplos están pensados para ser **reproducibles y fáciles de explorar**.
