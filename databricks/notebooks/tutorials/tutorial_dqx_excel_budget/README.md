
# Tutorial: Data Quality con DQX sobre Excel de Presupuesto

Este tutorial muestra cómo aplicar **Data Quality** de forma simple y escalable
sobre un Excel de presupuesto financiero usando **Databricks** y **DQX**.

## Caso de uso
Un Excel de presupuesto trimestral mantenido por Finanzas que necesita
validaciones antes de ser consumido por BI y reporting.

## Qué aprenderás
- Leer Excel con Spark
- Limpiar nombres de columnas
- Definir reglas de calidad en YAML
- Aplicar DQX para separar datos válidos y en cuarentena
- Implementar un patrón estándar de alertas

Pensado para demos y workshops de ~10 minutos.
