
# Data Quality con DQX sobre Excel de Presupuesto

Este tutorial muestra cómo aplicar **Data Quality** de forma estandarizada
sobre un Excel financiero usando **Databricks y DQX**.

## Qué se demuestra
- Lectura de Excel realista
- Limpieza de columnas
- Reglas desacopladas en YAML
- Separación Silver / Quarantine

Ideal para demos, workshops y webinars.
