# Qué encontrarás aca

## notebooks/tutorials

Ejemplos paso a paso usando notebooks de Databricks.

Incluyen casos como:

- Data Quality con **DQX**
- Ingestión de **Excel desde SharePoint con Lakeflow Connect**
- Uso de **Genie Code dentro de notebooks**
- Ejemplos de **gobernanza y permisos en Databricks**

Cada tutorial suele incluir:

- notebook principal
- README explicativo
- archivos de ejemplo cuando aplica

---

## lakeflow_jobs

Ejemplos simples de **Databricks Asset Bundles y Lakeflow Jobs**.

Incluyen:

- configuración del bundle (`databricks.yml`)
- definición del workflow
- notebook ejecutado como job

El ejemplo actual utiliza el dataset público:

`samples.nyctaxi.trips`

y muestra cómo ejecutar un análisis desde un job en Databricks.
